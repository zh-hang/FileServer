# FTPServer

这是一个文件服务器。使用C++ ThreadPool实现。

## 需求

### 服务端

- 支持客户端请求查询文件（文件名称，文件大小，更改日期），上传文件，下载文件，删除文件
- 支持多客户端并发向服务请求操作、
- 信令走TCP，数据传输支持TCP/UDP两种方式可选。 

### 客户端

- 查询文件列表
- 文件上传
- 文件下载
- 文件删除

### 待选需求

- 局域网内服务器发现（组播方式）
- 上传，下载支持断点上传和下载
- 支持用户安全机制：用户登录鉴权，权限控制。
- 数据传输走安全传输 SSL
